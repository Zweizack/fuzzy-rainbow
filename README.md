## { *Install* }
```bash
$ git clone https://github.com/Zweizack/fuzzy-rainbow.git
```
## { *Examples* }

#### RGB TO CMYK

```bash
$ python3 rgb_to_cmyk.py
```
```go
// CMYK: 0.0%, 0.0%, 0.0%, 49.8%
RGB: 128 128 128
```

#### CMYK TO RGB

```bash
$ python3 cmyk_to_rgb.py
```
```go
// RGB: 128, 128, 128
CMYK: 0 0 0 50
```

#### HEX TO RGB & CMYK
```bash
$ python3 hex.py
```
```go
// RGB:  (128,128,128)
// CMYK: 0.0%, 0.0%, 0.0%, 49.8%
Hex: 0x808080
```
